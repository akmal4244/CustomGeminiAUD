# CustomGeminiAUD

Laman slide interaktif untuk tatacara menggunakan Custom Gemini / Custom Gems bagi analisis dokumen pengauditan universiti awam.

## Live Site

https://akmal4244.github.io/CustomGeminiAUD/

## Kandungan

- Tatacara masuk ke Custom Gem
- Prompt pertama
- Prompt semakan tally Bahagian 3 dan Bahagian 5
- Prompt final lengkap
- Checklist output analisis yang betul

## Deployment

Laman ini menggunakan GitHub Pages daripada branch `main` dan folder `/root`.
