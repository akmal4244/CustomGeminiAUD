# CustomGeminiAUD

Laman slide interaktif bertajuk **Custom Gemini V4 untuk Analisis Laporan Audit** untuk melatih auditor menggunakan Custom Gemini, menyemak output analisis, dan memindahkan hasil ke template slide yang seragam.

## Live Site

https://akmal4244.github.io/CustomGeminiAUD/

## Kandungan

- Pengenalan pembentang dan tajuk latihan
- Agenda pembentangan
- Pautan dan butang terus ke Custom Gemini V4
- Arahan upload dokumen tajuk pengauditan, termasuk contoh UMK
- Full prompting standard dengan butang copy
- Semakan output 6 bahagian
- Bonus prompt soalan susulan
- Semakan auditor terhadap source file dokumen
- Pemindahan output ke template slide
- Rumusan visual menggunakan Gemini Image
- Penutup dan ucapan terima kasih

## Logo

Logo Jata Negara disimpan sebagai `assets/jata-negara.svg`, berdasarkan fail `Coat of arms of Malaysia.svg` daripada Wikimedia Commons.

Laman ini menggunakan GitHub Pages daripada branch `main` dan folder `/root`.
